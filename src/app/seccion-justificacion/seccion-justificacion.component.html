Justificación del framework

Angular es un framework de desarrollo web basado en TypeScript...

  • a)¿Para qué sirve?

    • Angular es un framework de desarrollo web de código abierto, mantenido por Google. Está diseñado para facilitar la creación de aplicaciones web de una sola página (SPA, por sus siglas en inglés) de manera eficiente y escalable. Angular permite a los desarrolladores crear aplicaciones dinámicas con un alto rendimiento, aprovechando características como la vinculación bidireccional de datos (two-way data binding), inyección de dependencias, y el enrutamiento integrado.

  • b)¿Qué aplicaciones tiene?

    • Angular se utiliza principalmente en el desarrollo de aplicaciones web interactivas y dinámicas. Es ideal para:

  • Aplicaciones web de una sola página (SPA)
  • Dashboards de administración.
  • Aplicaciones móviles (con Angular y Ionic).
  • Aplicaciones empresariales y de gestión de datos.

  • c)¿Sobre qué lenguajes se apoya?

    • Angular se basa principalmente en TypeScript, un superconjunto de JavaScript. TypeScript ofrece tipado estático, lo que ayuda a prevenir errores en tiempo de desarrollo. Angular también utiliza HTML y CSS para la creación de la interfaz de usuario, con extensiones específicas de Angular (como directivas y componentes).

  • d)¿Qué prerrequisitos necesita?

    • Para trabajar con Angular, se recomienda tener conocimienots básicos de:

  • JavaScript y TypeScript.
  • HTML y CSS.
  • Node.js y npm (gestor de paquetes de Node). Es útil también familiarizarse con conceptos de desarrollo de aplicaciones web, como el manejo de rutas y componentes
