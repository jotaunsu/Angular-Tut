Utilización del Framework

a)¿Cómo empezar a crar una aplicación base?

  1. Generar un componente:

    2. Para agregar nuevos compoentes, utiliza Angular CLI: ng generate component nombre-componente Esto crea un nuevo componente con su archivo HTML, CSS y TypeScript correspondiente.

  2. Modificar el compomente:Cambia el contenido del archivo nombre-compomente.compoment.html para definir la estrutucra de la interfaz de usuario.
  3. Incluir el compomente en la aplicación:Asegúrate de declarar el componente en el módulo correspondiente (app.module.ts) y de usar su selector enapp.component.html

b)¿Cómo visualizar los resultados de la ejecución?

Usa el servidor de desarrollo (con ng serve) para ver los cambios en tiempo real. Angular recargará la aplicación automáticamente cuando detecte modificaciones en los archivos.
