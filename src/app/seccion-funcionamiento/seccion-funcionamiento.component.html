Funcionamiento del Framework

a)¿Actúa sobre páginas, ficheros, secciones...?

Angular trabaja principalmente con componentes. Cada componente es responsable de una sección específica de la interfaz de usuario. Estos componentes pueden tener su propia lógica y diseño, y se organizan jerárquicamente dentro de la aplicación.

b)¿Cómo se editan/modifican los ficheros que actúan en el framework?

Los ficharos en Angular se modifican directamente en el código fuente del proyecto. Los componentes están compuestos por:

  1. HTML para la estructura de la página.
  2. CSS para los estilos.
  3. TypeScript para la lógica y el comportamiento.

c)¿Cómo se referencian los ficharos creados desde otras aplicciones (si es el caso)?

Si necesitas referenciar un componente, servicio o módulo desde otra aplicación, puedes exportar el componente desde un módulo y luego importarlo en el módulo de la aplicación que lo necesite.
