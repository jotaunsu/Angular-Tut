Primeros Pasos

Para crear una aplicación básica en angular que muestre "Hola mundo", sigue estos pasos:

  1. Crear un nuevo proyecto Angular.

    En la terminal, ejecuta:ng new hola-mundo

    Esto creará una nueva aplicación Angular con el nombre "hola-mundo".

  2. Navegar al directorio proyecto:

    cd hola-mundo

  3. Ejecutar el servidor de desarrollo:

    ng serve

    Esto iniciará el servidor y podrás ver tu aplicación en http://localhost:4200

  4. Modificar el archivo src/app/app.component.html para mostrar "Hola mundo":<>
  5. Ver los resultados:

    Al abrir http://localhost:4200 en tu navegador,verás el mensaje "Hola Mundo".
