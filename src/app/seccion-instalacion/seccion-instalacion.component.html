Instalación

    a)¿Se precisa instalación?

    Sí, Angular requiere instalación en el entorno de desarrollo local para poder crear y ejecutar aplicaciones.

    b)¿Qué pasos hay que realizar para realizar la instalación?

    1. Instalar Node.js y npm: Angular depende de Node.js, un entorno de ejecución de JavaScript, y npm, su gestor de paquetes.
      1. Descarga e instala Node.js desde nodejs.org
      2. npm se instala automáticamente con Node.js.
    2. Instalar Angular CLI (Command Line Interface):

      3. Abre un terminal o línea de comandos y ejecuta: npm install -g angular/cli.

    c)¿Versiones necesarias?

      Node.js: Se recomienda usar la versión LTS (Long Term Support) de Node.js.
      Angular CLI: Asegúrate de tener la última versión estable de Angualr CLI.

    d)¿Rutas para tener en cuenta?

    Angular CLI se instala globalmente, porlo que puedes ejecutarlo desde cualquier ruta del sistema.

    ¿Variables de entorno?

    Angular CLI usa algunas variables de entorno como el NODE_ENV para el modo de desarrollo o producción, pero no requiere configuraciones adicionales de entorno a menos que estés trabajando con herramientas específicas.

    e)¿Necesita base de datos?

    No, Angular no necesita una base de datos para funcionar, ya que es un framework frontend. Sin embargo, las aplicaciones Angular suelen consumir datos desde una base de datos mediante APIs (por ejemplo, REST o GraphQL).
